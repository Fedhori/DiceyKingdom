# 스킬 아이디어

스킬(Skill) 아이디어를 기록합니다.

## 아이디어 목록

- 분열 타격(Skill)
  - 기믹: 선택한 요원의 이번 턴 피해가 양옆 Situation에도 일부 전이
  - 의도: 광역 대응 수단 제공, 배치 가치 상승
  - 리스크: 다수 Situation 구간 난이도를 급격히 낮출 위험
- 은둔 현자 (Skill)
  - 효과: 1이 나온 주사위에만 사용 가능, 해당 주사위 +3
  - 쿨다운: 2
  - 플레이버: 그를 아는 이는 적으나, 그가 쓴 글은 세상을 움직인다.
- 첩 (Skill)
  - 효과: 선택한 주사위 리롤
  - 쿨다운: 1
  - 플레이버: 그녀에게는 여러 이름이 있다. 장막 뒤의 단검. 왕의 세 번째 눈. 침실의 스킬.
- 배치 전환관 (Skill)
  - 기믹: `P4`에서 서로 다른 `Situation`에 배치된 주사위 2개의 위치를 교환
  - 의도: `P2` 배치 실수의 복구 수단 제공 + 고급 전술 선택지 추가
  - 리스크: 기본 규칙의 "상황 잠금" 긴장감을 약화할 수 있음
  - 개선안:
    - 쿨다운/사용 조건을 부여해 상시 재배치화를 방지
    - 같은 턴 내 1회 또는 상황당 1회 제한
- 무쿨다운 비용형 Skill
  - 기믹: 쿨다운 대신 골드를 소모해 즉시 효과 발동
  - 의도: 비상 상황에서 골드를 전술 파워로 교환하는 선택 제공
  - 리스크: 골드 수급이 안정되면 사실상 상시 버프로 변질될 수 있음
  - 개선안:
    - 글로벌 하드 제한 대신, 개별 스킬별 비용 구조로 위험도 제어
    - 예시: `용병단장`은 사용 시마다 비용이 1씩 증가
    - 특정 태그 상황에서만 사용 가능하게 설계해 범용성 과다를 방지
- 용병단장 (Skill)
  - 기믹: 선택한 주사위 눈 +1, 사용 시 골드 소모
  - 의도: 골드를 즉시 전투력으로 치환하는 전술형 버튼
  - 리스크: 골드 수급과 결합 시 반복 사용 효율이 과도해질 수 있음
  - 개선안:
    - 사용할수록 비용 증가(누진)
    - 특정 턴/상황에서만 추가 보정 가능
- 스킬 용어 통일(영문 우선)
  - 기믹: 스킬을 "인물" 테마로 묶는 명명 규칙 적용
  - 용어: `Skill`
  - 의도: 세계관 몰입과 인지 일관성 강화
  - 리스크: 용어가 추상적이면 기능 직관성이 떨어질 수 있음
  - 개선안:
    - 카테고리명은 `Skill`로 고정
    - 카드 타이틀/아이콘에서 기능 타입을 보조 표기
- 스킬 슬롯 가중치 시스템
  - 기믹: 스킬마다 점유 슬롯이 다르고, 플레이어는 총 슬롯 한도 내에서 구성
  - 의도: 단순 개수 제한보다 빌드 의사결정 밀도 강화
  - 리스크: UI/인지 복잡도 증가, 초반 진입장벽 상승
  - 개선안:
    - 초반에는 슬롯 비용 1 위주로 시작해 점진적으로 고비용 스킬 해금
    - 덱빌딩 UI처럼 현재 점유/잔여 슬롯을 항상 시각화
