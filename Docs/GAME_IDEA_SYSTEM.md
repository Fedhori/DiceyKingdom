# 시스템 아이디어

시스템 전반(런 구조, 자원 구조, 전투 기조)에 해당하는 아이디어를 기록합니다.

## 아이디어 목록

- 안정도(왕국 생존 자원) 시스템
  - 0 이하가 되면 멸망(게임 오버)
  - 상황 실패/디버프로 감소, 보상/행동으로 회복
- 주사위 분배 기반 해결 구조
  - 매 턴 보유 주사위를 상황 카드 슬롯에 할당 후 굴림
  - 상황마다 주사위 최대 할당 개수 제한 가능
- 주사위 강화 시스템
  - 주사위당 강화 1개 장착
  - 예시: 모든 눈 +1, 리롤 1회 보장, 2회 굴려 높은 눈 사용
- 스킬(Skill) 시스템
  - 액티브 효과 + 턴 쿨다운
  - 전술적 보정(리롤/눈 보정/남은 턴수 조절 등)
- 구조물(Structure) 시스템
  - 런 동안 유지되는 영구 패시브
  - 빌드 방향과 장기 생존력 강화
- 오버킬 매커니즘
  - 슬롯을 초과해서 달성한 수치(오버킬)에 반응하는 트리거/보상 시스템
  - 방향성: 고점 주사위 운용과 시너지 빌드의 보상 체감 강화
  - 세부 규칙은 미정(즉시 보상형 vs 누적형 vs 조건형)

## 진행 예시 메모

- 플레이어 기본 상태 예시:
  - 주사위 4개(기본 3, 강화 1)
  - 스킬 3개(리롤 CD1, 눈 +1 CD1, 상황 약화 -2 CD2)
  - 구조물 1개(이번 턴 최저 눈 +1)
- 전술 포인트:
  - 단기적으로 실패를 허용하고 핵심 실패를 지연시키는 선택
  - 고가치 상황 우선 해결 vs 안정도 보호용 행동 선택
