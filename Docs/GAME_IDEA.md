# 게임 아이디어

이 문서는 게임 아이디어를 단일 파일로 기록합니다.
구조화된 현재 설계 기준은 `Docs/GAME_STRUCTURE.md`를 우선합니다.

## 기록 규칙

- 이 문서는 아이디어 후보와 메모만 기록합니다.
- 사용자 지시가 없는 의사결정 기록(채택/보류/기각 과정)은 남기지 않습니다.
- 밸런스 수치는 초안이며, 플레이테스트로 조정합니다.
- 핵심 파라미터 확정 전에는 수치 대신 기믹/조건/리스크를 우선 기록합니다.

### 아이디어 기록 템플릿

```md
- (이름)
  - 의도: (왜 필요한 요소인지)
  - 기대 재미: (플레이어가 어떤 판단/쾌감을 얻는지)
  - 밸런스 영향: (초반/중반/후반, 오버/언더 가능성)
  - 리스크: (악용 루프, 조합 강제, 의미 없는 선택 등)
  - 개선안: (기믹 조정, 조건 추가, 제약 부여 등)
```

## 시스템 아이디어

- 안정도(왕국 생존 자원) 시스템
  - 0 이하가 되면 멸망(게임 오버)
  - 상황 실패/디버프로 감소, 보상/행동으로 회복
- 주사위 분배 기반 해결 구조
  - 매 턴 보유 주사위를 상황 카드 슬롯에 할당 후 굴림
  - 상황마다 주사위 최대 할당 개수 제한 가능
- 주사위 강화 시스템
  - 주사위당 강화 1개 장착
  - 예시: 모든 눈 +1, 리롤 1회 보장, 2회 굴려 높은 눈 사용
- 스킬(Skill) 시스템
  - 액티브 효과 + 턴 쿨다운
  - 전술적 보정(리롤/눈 보정/남은 턴수 조절 등)
- 구조물(Structure) 시스템
  - 런 동안 유지되는 영구 패시브
  - 빌드 방향과 장기 생존력 강화
- 오버킬 매커니즘
  - 슬롯을 초과해서 달성한 수치(오버킬)에 반응하는 트리거/보상 시스템
  - 방향성: 고점 주사위 운용과 시너지 빌드의 보상 체감 강화
  - 세부 규칙은 미정(즉시 보상형 vs 누적형 vs 조건형)

## 상황 아이디어

- 메뚜기 떼
  - 타입: 기근
  - 남은 턴수: 2
  - 목표치: 5, 8
  - 실패: 안정도 -1, 3턴 `굶주림` 디버프
  - 굶주림: 기근 상황 추가 실패 시 안정도 -1 추가
  - 성공: 안정도 +1
- 오크 부대
  - 타입: 침략
  - 남은 턴수: 1
  - 목표치: 10 (최대 주사위 3개)
  - 실패: 안정도 -2
  - 성공: 골드 +3
- 영지 순찰
  - 타입: 내정
  - 남은 턴수 없음(상시 행동칸)
  - 목표치: 10
  - 완료: 안정도 +1
  - 완료 후 재생성
- 거대 프로젝트
  - 목표치: 50
  - 턴 제한 없음
  - 성공 보상: `웅장한 동상` 패시브 획득
- 반란군
  - 기믹: 방치할수록 요구 목표치가 누적 증가하는 위기 카드
  - 의도: 방치 시 리스크가 누적되는 고우선 상황 카드
  - 기대 재미: "지금 처리할지, 다른 위기를 먼저 막을지" 우선순위 판단 압박
  - 밸런스 영향: 조기 대응 시 부담이 낮고, 지연할수록 비용이 커지는 선택 구조를 형성
  - 리스크: 카드를 늦게 뽑았을 때 사실상 해결 불가 상태로 커질 수 있음
  - 개선안:
    - 초기 부담을 낮게 설정해 "빠르게 처리할수록 이득" 구조를 보장
    - 누적 증가 완화 장치(상한, 감소 트리거, 일시 정지 효과) 중 1개 이상 포함
    - 대응 수단(Decree/Skill/Structure) 접근성을 확보해 불합리 누적을 방지
- 내부 공작망
  - 기믹: 이 상황이 생성될 때, 스킬 풀 전체에서 무작위 스킬 1개를 "사용 불가"로 봉인
  - 해제 조건: 해당 상황을 해결할 때에만 봉인 해제
  - 의도: 스킬 의존 빌드에 대한 카운터 압박 + 우선순위 전환 강제
  - 기대 재미: "핵심 카드를 살릴지, 다른 위기를 먼저 막을지"의 고통스러운 선택
  - 밸런스 영향: 특정 빌드의 고점을 눌러 평균 난이도를 높일 수 있음
  - 리스크:
    - 핵심 스킬 봉인 시 대응 수단이 사라져 런 파괴감이 커질 수 있음
    - 운 나쁨 체감이 강해지면 공정성 인식이 빠르게 악화될 수 있음
  - 개선안:
    - 봉인 대상 정보(누가 봉인됐는지/해제 조건)를 카드와 HUD에 상시 노출
    - 봉인/해제 로그를 턴 로그에 남겨 피격 원인을 추적 가능하게 유지

## 요원 아이디어

- 묘지기(요원)
  - 기믹: 배정된 적이 이미 처치된 경우, 다음 적으로 자동 재지정 후 공격
  - 의도: 과잉 화력으로 인한 공중분해 손실을 부분 완화
  - 리스크: 배치 순서 전략의 의미를 약화하거나 난이도 하향으로 이어질 수 있음
- 선봉대(요원/무기)
  - 기믹: 해당 Situation을 가장 먼저 대결하면 버프 획득
  - 의도: 배치/공격 순서에 의미를 부여하고 선공 선택의 재미 강화
  - 리스크: 선공 보너스가 과하면 사실상 고정 오프너가 강제될 수 있음
- 협공가(요원)
  - 기믹: 같은 Situation을 자신보다 먼저 대결한 요원 수에 비례해 버프 획득
  - 의도: 단일 대상 집중 화력 빌드의 시너지 강화
  - 리스크: 다인 집중이 정답화되면 타깃 분산 전략이 사라질 수 있음
- 마무리 사냥꾼(요원/무기)
  - 기믹: 대상 Situation의 남은 주사위가 절반 이하일 때 버프 획득
  - 의도: 전투 중반 이후의 타깃 전환 판단을 유도
  - 리스크: 잔량 임계값 확인 피로가 누적될 수 있음
- 현상금 집행자(요원)
  - 기믹: 막타 처치 시 소량 골드 획득
  - 의도: 타격 순서 최적화 보상 제공
  - 리스크: 막타 몰아주기 메타가 과해지면 플레이 다양성이 감소

## 스킬 아이디어

- 분열 타격(Skill)
  - 기믹: 선택한 요원의 이번 턴 피해가 양옆 Situation에도 일부 전이
  - 의도: 광역 대응 수단 제공, 배치 가치 상승
  - 리스크: 다수 Situation 구간 난이도를 급격히 낮출 위험
- 은둔 현자 (Skill)
  - 효과: 1이 나온 주사위에만 사용 가능, 해당 주사위 +3
  - 쿨다운: 2
  - 플레이버: 그를 아는 이는 적으나, 그가 쓴 글은 세상을 움직인다.
- 첩 (Skill)
  - 효과: 선택한 주사위 리롤
  - 쿨다운: 1
  - 플레이버: 그녀에게는 여러 이름이 있다. 장막 뒤의 단검. 왕의 세 번째 눈. 침실의 스킬.
- 배치 전환관 (Skill)
  - 기믹: `P4`에서 서로 다른 `Situation`에 배치된 주사위 2개의 위치를 교환
  - 의도: `P2` 배치 실수의 복구 수단 제공 + 고급 전술 선택지 추가
  - 리스크: 기본 규칙의 "상황 잠금" 긴장감을 약화할 수 있음
  - 개선안:
    - 쿨다운/사용 조건을 부여해 상시 재배치화를 방지
    - 같은 턴 내 1회 또는 상황당 1회 제한
- 무쿨다운 비용형 Skill
  - 기믹: 쿨다운 대신 골드를 소모해 즉시 효과 발동
  - 의도: 비상 상황에서 골드를 전술 파워로 교환하는 선택 제공
  - 리스크: 골드 수급이 안정되면 사실상 상시 버프로 변질될 수 있음
  - 개선안:
    - 글로벌 하드 제한 대신, 개별 스킬별 비용 구조로 위험도 제어
    - 예시: `용병단장`은 사용 시마다 비용이 1씩 증가
    - 특정 태그 상황에서만 사용 가능하게 설계해 범용성 과다를 방지
- 용병단장 (Skill)
  - 기믹: 선택한 주사위 눈 +1, 사용 시 골드 소모
  - 의도: 골드를 즉시 전투력으로 치환하는 전술형 버튼
  - 리스크: 골드 수급과 결합 시 반복 사용 효율이 과도해질 수 있음
  - 개선안:
    - 사용할수록 비용 증가(누진)
    - 특정 턴/상황에서만 추가 보정 가능
- 스킬 용어 통일(영문 우선)
  - 기믹: 스킬을 "인물" 테마로 묶는 명명 규칙 적용
  - 용어: `Skill`
  - 의도: 세계관 몰입과 인지 일관성 강화
  - 리스크: 용어가 추상적이면 기능 직관성이 떨어질 수 있음
  - 개선안:
    - 카테고리명은 `Skill`로 고정
    - 카드 타이틀/아이콘에서 기능 타입을 보조 표기
- 스킬 슬롯 가중치 시스템
  - 기믹: 스킬마다 점유 슬롯이 다르고, 플레이어는 총 슬롯 한도 내에서 구성
  - 의도: 단순 개수 제한보다 빌드 의사결정 밀도 강화
  - 리스크: UI/인지 복잡도 증가, 초반 진입장벽 상승
  - 개선안:
    - 초반에는 슬롯 비용 1 위주로 시작해 점진적으로 고비용 스킬 해금
    - 덱빌딩 UI처럼 현재 점유/잔여 슬롯을 항상 시각화

## 구조물 아이디어

- 국경수비대 (Structure)
  - 효과: 모든 `침략` 태그 상황의 시작 목표치 -1
- 최후의 의지 (Structure)
  - 기믹: 안정도가 0 아래로 내려가도 1턴 유예 상태로 생존
  - 의도: 역전 기회 제공, 극단 턴에서의 드라마 강화, 더 과감한 리스크 선택 유도
  - 리스크: 사실상 추가 목숨처럼 작동해 난이도 체감을 왜곡할 수 있음
  - 개선안:
    - 획득 난이도를 높게 설정해 접근성을 제한
    - 런당 1회 한정
    - 유예 턴 종료 시 안정도 1 이상 회복 실패하면 즉시 게임 오버
- 은행 (Structure)
  - 기믹: 골드를 `-N`까지 대출(적자) 가능하게 확장
  - 의도: 단기 부채를 통한 고위험 고보상 운영 허용
  - 리스크: 대출이 너무 강하면 사실상 공짜 골드로 변질
  - 개선안:
    - 효과 텍스트는 단일 규칙(대출 한도)로 유지해 직관성을 우선
    - UI에 현재 골드와 대출 한도(`-N`)를 상시 표시
    - 중복 획득 시 대출 한도는 중복 적용(누적)
- 구조물 용어 전환(영문 우선)
  - 기믹: "유산"을 구조물 명칭으로 교체
  - 용어: `Structure`
  - 의도: 왕국 경영 판타지와 시스템 의미를 더 직관적으로 연결
  - 리스크: 용어가 너무 일반적이면 스킬/칙령과 경계가 흐려질 수 있음
  - 개선안:
    - 카테고리 용어 + 서브타입(예: Military, Civic, Faith) 동시 표기
- 웅장한 동상 (패시브)
  - 효과: 매 3턴마다 안정도 +1
  - 획득처: 거대 프로젝트 완료 보상

## 칙령 아이디어

- 용병단 고용서 (Decree)
  - 대상: `침략` 태그 상황만 사용 가능
  - 효과: 해당 상황 목표치 -4
  - 테마: 칙령/문서형 소모품

## 진행 예시 메모

- 플레이어 기본 상태 예시:
  - 주사위 4개(기본 3, 강화 1)
  - 스킬 3개(리롤 CD1, 눈 +1 CD1, 상황 약화 -2 CD2)
  - 구조물 1개(이번 턴 최저 눈 +1)
- 전술 포인트:
  - 단기적으로 실패를 허용하고 핵심 실패를 지연시키는 선택
  - 고가치 상황 우선 해결 vs 안정도 보호용 행동 선택

## 오픈 질문

- 스테이지 단위 + 준비시간 액션 구조를 2차 실험에 언제 전환할지
- 2차 사이클 난이도 상승 규칙(신규 생성, 상황 강화, 액션 압박 중 우선 축)
- v0 프리셋 3개의 실제 조합안 확정
- 추가 상황 2종(`bandit_smugglers`, `siege_shaman`)의 수치 튜닝 기준 확정
- 타입 태그별 전용 디버프/보상 풀 구성
- 성장 보상 주기(몇 턴마다 강화/스킬/구조물 선택 기회 제공할지)
- 골드의 사용처(상점, 즉시 회복, 확률 이벤트 등)
- 오버킬 보상 규칙(남는 수치 이월, 추가 골드 획득, 전용 트리거 여부)
- 증가형 상황 위협의 누적 강도 곡선과 완화 장치(상한/감소/정지) 기준
- 은행형 대출 한도(`N`) 값 확정
