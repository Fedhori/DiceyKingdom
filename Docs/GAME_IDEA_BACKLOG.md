# 게임 아이디어 백로그

이 문서는 떠오르는 아이디어를 빠르게 기록하고, 구현 전 후보를 정리하기 위한 백로그입니다.
구조화된 현재 설계 기준은 `Docs/GAME_STRUCTURE.md`를 우선합니다.

## 기록 규칙

- 상태 태그:
  - `[아이디어]` 아직 검토 전
  - `[검토중]` 시스템 규칙과 충돌 여부 확인 중
  - `[채택]` 현재 기획 기준에 반영됨
  - `[보류]` 재미는 있으나 우선순위 낮음
- 밸런스 수치는 초안이며, 플레이테스트로 조정합니다.

## 시스템 아이디어

- `[채택]` 안정도(왕국 체력) 시스템
  - 0 이하가 되면 멸망(게임 오버)
  - 상황 실패/디버프로 감소, 보상/행동으로 회복
- `[채택]` 주사위 분배 기반 해결 구조
  - 매 턴 보유 주사위를 상황 카드 슬롯에 할당 후 굴림
  - 상황마다 주사위 최대 할당 개수 제한 가능
- `[채택]` 주사위 강화 시스템
  - 주사위당 강화 1개 장착
  - 예시: 모든 눈 +1, 리롤 1회 보장, 2회 굴려 높은 눈 사용
- `[채택]` 스킬 시스템
  - 액티브 효과 + 턴 쿨다운
  - 전술적 보정(리롤/눈 보정/남은 턴수 조절 등)
- `[채택]` 유산 시스템
  - 런 동안 유지되는 영구 패시브
  - 빌드 방향과 장기 생존력 강화
- `[아이디어]` 오버킬 매커니즘
  - 슬롯을 초과해서 달성한 수치(오버킬)에 반응하는 트리거/보상 시스템
  - 방향성: 고점 주사위 운용과 시너지 빌드의 보상 체감 강화
  - 세부 규칙은 미정(즉시 보상형 vs 누적형 vs 조건형)

## 상황 카드 아이디어

- `[채택]` 메뚜기 떼
  - 타입: 기근
  - 남은 턴수: 2
  - 목표치: 5, 8
  - 실패: 안정도 -1, 3턴 `굶주림` 디버프
  - 굶주림: 기근 상황 추가 실패 시 안정도 -1 추가
  - 성공: 안정도 +1

- `[채택]` 오크 부대
  - 타입: 침략
  - 남은 턴수: 1
  - 목표치: 10 (최대 주사위 3개)
  - 실패: 안정도 -2
  - 성공: 골드 +3

- `[채택]` 영지 순찰
  - 타입: 내정
  - 남은 턴수 없음(상시 행동칸)
  - 목표치: 10
  - 완료: 안정도 +1
  - 완료 후 재생성

- `[아이디어]` 거대 프로젝트
  - 목표치: 50
  - 턴 제한 없음
  - 성공 보상: `웅장한 동상` 패시브 획득

## 스킬/유산/아이템 아이디어

- `[아이디어]` 은둔 현자 (스킬)
  - 효과: 1이 나온 주사위에만 사용 가능, 해당 주사위 +3
  - 쿨다운: 2
  - 플레이버: 그를 아는 이는 적으나, 그가 쓴 글은 세상을 움직인다.

- `[아이디어]` 첩 (스킬)
  - 효과: 선택한 주사위 리롤
  - 쿨다운: 1
  - 플레이버: 그녀에게는 여러 이름이 있다. 장막 뒤의 단검. 왕의 세 번째 눈. 침실의 조언자.

- `[아이디어]` 국경수비대 (유산)
  - 효과: 모든 `침략` 태그 상황의 시작 목표치 -1

- `[아이디어]` 용병단 고용서 (일회성 아이템)
  - 대상: `침략` 태그 상황만 사용 가능
  - 효과: 해당 상황 목표치 -4
  - 테마: 칙령/문서형 소모품

- `[아이디어]` 웅장한 동상 (패시브)
  - 효과: 매 3턴마다 안정도 +1
  - 획득처: 거대 프로젝트 완료 보상

## 진행 예시 메모

- 플레이어 기본 상태 예시:
  - 주사위 4개(기본 3, 강화 1)
  - 스킬 1개(상황 남은 턴수 +1, 쿨다운 4, 상황당 1회)
  - 유산 1개(이번 턴 최저 눈 +1)
- 전술 포인트:
  - 단기적으로 실패를 허용하고 핵심 실패를 지연시키는 선택
  - 고가치 상황 우선 해결 vs 안정도 보호용 행동 선택

## 오픈 질문

- 턴당 신규 상황 카드 수와 난이도 상승 규칙
- 타입 태그별 전용 디버프/보상 풀 구성
- 성장 보상 주기(몇 턴마다 강화/스킬/유산 선택 기회 제공할지)
- 골드의 사용처(상점, 즉시 회복, 확률 이벤트 등)
- 오버킬 보상 규칙(남는 수치 이월, 추가 자원 획득, 전용 트리거 여부)
