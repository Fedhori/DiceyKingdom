# 게임 아이디어 백로그

이 문서는 떠오르는 아이디어를 빠르게 기록하고, 구현 전 후보를 정리하기 위한 백로그입니다.
구조화된 현재 설계 기준은 `Docs/GAME_STRUCTURE.md`를 우선합니다.

## 기록 규칙

- 상태 태그:
  - `[아이디어]` 아직 검토 전
  - `[검토중]` 시스템 규칙과 충돌 여부 확인 중
  - `[채택]` 현재 기획 기준에 반영됨
  - `[보류]` 재미는 있으나 우선순위 낮음
- 밸런스 수치는 초안이며, 플레이테스트로 조정합니다.
- 핵심 파라미터 확정 전에는 수치 대신 기믹/조건/리스크를 우선 기록합니다.

### 아이디어 기록 템플릿

```md
- `[아이디어]` (이름)
  - 의도: (왜 필요한 요소인지)
  - 기대 재미: (플레이어가 어떤 판단/쾌감을 얻는지)
  - 밸런스 영향: (초반/중반/후반, 오버/언더 가능성)
  - 리스크: (악용 루프, 조합 강제, 의미 없는 선택 등)
  - 개선안: (기믹 조정, 조건 추가, 제약 부여 등)
  - 상태: `[아이디어]` / `[검토중]` / `[채택]` / `[보류]`
```

## 시스템 아이디어

- `[채택]` 안정도(왕국 체력) 시스템
  - 0 이하가 되면 멸망(게임 오버)
  - 상황 실패/디버프로 감소, 보상/행동으로 회복
- `[채택]` 주사위 분배 기반 해결 구조
  - 매 턴 보유 주사위를 상황 카드 슬롯에 할당 후 굴림
  - 상황마다 주사위 최대 할당 개수 제한 가능
- `[채택]` 주사위 강화 시스템
  - 주사위당 강화 1개 장착
  - 예시: 모든 눈 +1, 리롤 1회 보장, 2회 굴려 높은 눈 사용
- `[채택]` 전령(Advisor) 시스템
  - 액티브 효과 + 턴 쿨다운
  - 전술적 보정(리롤/눈 보정/남은 턴수 조절 등)
- `[채택]` 구조물(Structure) 시스템
  - 런 동안 유지되는 영구 패시브
  - 빌드 방향과 장기 생존력 강화
- `[아이디어]` 오버킬 매커니즘
  - 슬롯을 초과해서 달성한 수치(오버킬)에 반응하는 트리거/보상 시스템
  - 방향성: 고점 주사위 운용과 시너지 빌드의 보상 체감 강화
  - 세부 규칙은 미정(즉시 보상형 vs 누적형 vs 조건형)

## 상황 카드 아이디어

- `[채택]` 메뚜기 떼
  - 타입: 기근
  - 남은 턴수: 2
  - 목표치: 5, 8
  - 실패: 안정도 -1, 3턴 `굶주림` 디버프
  - 굶주림: 기근 상황 추가 실패 시 안정도 -1 추가
  - 성공: 안정도 +1

- `[채택]` 오크 부대
  - 타입: 침략
  - 남은 턴수: 1
  - 목표치: 10 (최대 주사위 3개)
  - 실패: 안정도 -2
  - 성공: 골드 +3

- `[채택]` 영지 순찰
  - 타입: 내정
  - 남은 턴수 없음(상시 행동칸)
  - 목표치: 10
  - 완료: 안정도 +1
  - 완료 후 재생성

- `[아이디어]` 거대 프로젝트
  - 목표치: 50
  - 턴 제한 없음
  - 성공 보상: `웅장한 동상` 패시브 획득

- `[아이디어]` 반란군
  - 기믹: 방치할수록 요구 목표치가 누적 증가하는 위기 카드
  - 의도: 방치 시 리스크가 누적되는 고우선 상황 카드
  - 기대 재미: "지금 처리할지, 다른 위기를 먼저 막을지" 우선순위 판단 압박
  - 밸런스 영향: 조기 대응 시 부담이 낮고, 지연할수록 비용이 커지는 선택 구조를 형성
  - 리스크: 카드를 늦게 뽑았을 때 사실상 해결 불가 상태로 커질 수 있음
  - 개선안:
    - 초기 부담을 낮게 설정해 "빠르게 처리할수록 이득" 구조를 보장
    - 누적 증가 완화 장치(상한, 감소 트리거, 일시 정지 효과) 중 1개 이상 포함
    - 대응 수단(Decree/Advisor/Structure) 접근성을 확보해 불합리 누적을 방지
  - 상태: `[아이디어]`

- `[아이디어]` 내부 공작망
  - 기믹: 이 상황이 생성될 때, 전령 풀 전체에서 무작위 전령 1개를 "사용 불가"로 봉인
  - 해제 조건: 해당 상황을 해결할 때에만 봉인 해제
  - 의도: 전령 의존 빌드에 대한 카운터 압박 + 우선순위 전환 강제
  - 기대 재미: "핵심 카드를 살릴지, 다른 위기를 먼저 막을지"의 고통스러운 선택
  - 밸런스 영향: 특정 빌드의 고점을 눌러 평균 난이도를 높일 수 있음
  - 리스크:
    - 핵심 전령 봉인 시 대응 수단이 사라져 런 파괴감이 커질 수 있음
    - 운 나쁨 체감이 강해지면 공정성 인식이 빠르게 악화될 수 있음
  - 개선안:
    - 봉인 대상 정보(누가 봉인됐는지/해제 조건)를 카드와 HUD에 상시 노출
    - 봉인/해제 로그를 턴 로그에 남겨 피격 원인을 추적 가능하게 유지
  - 상태: `[아이디어]`

## 전령/구조물/칙령 아이디어

- `[아이디어]` 은둔 현자 (Advisor)
  - 효과: 1이 나온 주사위에만 사용 가능, 해당 주사위 +3
  - 쿨다운: 2
  - 플레이버: 그를 아는 이는 적으나, 그가 쓴 글은 세상을 움직인다.

- `[아이디어]` 첩 (Advisor)
  - 효과: 선택한 주사위 리롤
  - 쿨다운: 1
  - 플레이버: 그녀에게는 여러 이름이 있다. 장막 뒤의 단검. 왕의 세 번째 눈. 침실의 전령.

- `[아이디어]` 배치 전환관 (Advisor)
  - 기믹: `P4`에서 서로 다른 `Situation`에 배치된 주사위 2개의 위치를 교환
  - 의도: `P2` 배치 실수의 복구 수단 제공 + 고급 전술 선택지 추가
  - 리스크: 기본 규칙의 "상황 잠금" 긴장감을 약화할 수 있음
  - 개선안:
    - 쿨다운/사용 조건을 부여해 상시 재배치화 방지
    - 같은 턴 내 1회 또는 상황당 1회 제한
  - 상태: `[아이디어]`

- `[아이디어]` 무쿨다운 비용형 Advisor
  - 기믹: 쿨다운 대신 골드를 소모해 즉시 효과 발동
  - 의도: 비상 상황에서 골드를 전술 파워로 교환하는 선택 제공
  - 리스크: 골드 수급이 안정되면 사실상 상시 버프가 될 수 있음
  - 개선안:
    - 글로벌 하드 제한 대신, 개별 전령별 비용 구조로 위험도 제어
    - 예시: `용병단장`은 사용 시마다 비용이 1씩 증가
    - 특정 태그 상황에서만 사용 가능하게 설계해 범용성 과다를 방지
  - 상태: `[아이디어]`

- `[아이디어]` 용병단장 (Advisor)
  - 기믹: 선택한 주사위 눈 +1, 사용 시 골드 소모
  - 의도: 골드를 즉시 전투력으로 치환하는 전술형 버튼
  - 리스크: 골드 수급과 결합 시 반복 사용 효율이 과도해질 수 있음
  - 개선안:
    - 사용할수록 비용 증가(누진)
    - 특정 턴/상황에서만 추가 보정 가능
  - 상태: `[아이디어]`

- `[채택]` 전령 용어 통일(영문 우선)
  - 기믹: 전령을 "인물" 테마로 묶는 명명 규칙 적용
  - 확정 용어: `Advisor`
  - 의도: 세계관 몰입과 인지 일관성 강화
  - 리스크: 용어가 추상적이면 기능 직관성이 떨어질 수 있음
  - 개선안:
    - 카테고리명은 `Advisor`로 고정하고
    - 카드 타이틀/아이콘에서 기능 타입을 보조 표기
  - 상태: `[채택]`

- `[아이디어]` 전령 슬롯 가중치 시스템
  - 기믹: 전령마다 점유 슬롯이 다르고, 플레이어는 총 슬롯 한도 내에서 구성
  - 의도: 단순 개수 제한보다 빌드 의사결정 밀도 강화
  - 리스크: UI/인지 복잡도 증가, 초반 진입장벽 상승
  - 개선안:
    - 초반에는 슬롯 비용 1 위주로 시작해 점진적으로 고비용 전령 해금
    - 덱빌딩 UI처럼 현재 점유/잔여 슬롯을 항상 시각화
  - 상태: `[아이디어]`

- `[아이디어]` 국경수비대 (Structure)
  - 효과: 모든 `침략` 태그 상황의 시작 목표치 -1

- `[아이디어]` 최후의 의지 (Structure)
  - 기믹: 안정도가 0 아래로 내려가도 1턴 유예 상태로 생존
  - 의도: 역전 기회 제공, 극단 턴에서의 드라마 강화, 더 과감한 리스크 선택 유도
  - 리스크: 사실상 추가 목숨처럼 작동해 난이도 체감을 왜곡할 수 있음
  - 개선안:
    - 획득 난이도를 높게 설정해 접근성을 제한
    - 런당 1회 한정
    - 유예 턴 종료 시 안정도 1 이상 회복 실패하면 즉시 게임 오버
  - 상태: `[아이디어]`

- `[아이디어]` 은행 (Structure)
  - 기믹: 골드를 `-N`까지 대출(적자) 가능하게 확장
  - 의도: 단기 부채를 통한 고위험 고보상 운영 허용
  - 리스크: 대출이 너무 강하면 사실상 공짜 골드로 변질
  - 개선안:
    - 효과 텍스트는 단일 규칙(대출 한도)로 유지해 직관성을 우선
    - UI에 현재 골드와 대출 한도(`-N`)를 상시 표시
    - 중복 획득 시 대출 한도는 중복 적용(누적)
  - 상태: `[검토중]`

- `[채택]` 구조물 용어 전환(영문 우선)
  - 기믹: "유산"을 구조물 명칭으로 교체
  - 확정 용어: `Structure`
  - 의도: 왕국 경영 판타지와 시스템 의미를 더 직관적으로 연결
  - 리스크: 용어가 너무 일반적이면 전령/칙령과 경계가 흐려질 수 있음
  - 개선안:
    - 카테고리 용어 + 서브타입(예: Military, Civic, Faith) 동시 표기
  - 상태: `[채택]`

- `[아이디어]` 용병단 고용서 (Decree)
  - 대상: `침략` 태그 상황만 사용 가능
  - 효과: 해당 상황 목표치 -4
  - 테마: 칙령/문서형 소모품

- `[아이디어]` 웅장한 동상 (패시브)
  - 효과: 매 3턴마다 안정도 +1
  - 획득처: 거대 프로젝트 완료 보상

## 진행 예시 메모

- 플레이어 기본 상태 예시:
  - 주사위 4개(기본 3, 강화 1)
  - 전령 1개(상황 남은 턴수 +1, 쿨다운 4, 상황당 1회)
  - 구조물 1개(이번 턴 최저 눈 +1)
- 전술 포인트:
  - 단기적으로 실패를 허용하고 핵심 실패를 지연시키는 선택
  - 고가치 상황 우선 해결 vs 안정도 보호용 행동 선택

## 오픈 질문

- 턴당 신규 상황 카드 수와 난이도 상승 규칙
- 타입 태그별 전용 디버프/보상 풀 구성
- 성장 보상 주기(몇 턴마다 강화/전령/구조물 선택 기회 제공할지)
- 골드의 사용처(상점, 즉시 회복, 확률 이벤트 등)
- 오버킬 보상 규칙(남는 수치 이월, 추가 골드 획득, 전용 트리거 여부)
- 증가형 상황 카드의 누적 강도 곡선과 완화 장치(상한/감소/정지) 기준
- 무쿨다운 비용형 Advisor의 안전장치(누진 비용/사용 조건/대체 비용) 기준
- 은행형 대출 한도(`N`) 값 확정
