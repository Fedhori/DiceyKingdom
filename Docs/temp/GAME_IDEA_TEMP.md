# GAME IDEA (TEMP)

이 문서는 머릿속에 흩어진 아이디어 파편을 하나의 기획 초안으로 묶어 기록하는 임시 정리본이다.

## 장르/레퍼런스

- 장르: 인력 관리 중심 로그라이크
- 레퍼런스: 다키스트 던전, XCOM

## 다키스트 던전과의 차별 포인트

- 다키스트 던전보다 런 길이와 임무 단위를 더 짧고 가볍게 설계한다.
- 인력 관리(모집/배치/피로/손실) 중심 재미를 더 전면에 둔다.
- 로그라이크 특성인 실험과 재시도를 더 빠르게 반복 가능한 구조로 만든다.

## 코어 컨셉/테마/승리 구조

- 컨셉: 모험가를 모집해 외부 위협으로부터 왕국을 수호한다.
- 테마: 중세 다크 판타지
- 승리 구조:
- 주기적으로 위기가 찾아오고, 높은 난이도와 다수 임무로 강한 인력 부족 압박을 준다.
- 마지막 위기에서 최고 난이도 도전을 돌파하면 승리한다.

## 게임 구성 요소

### 1) 모험가

- 역할: XCOM의 병사, 다키스트 던전의 영웅과 유사한 핵심 인력 유닛
- 임무 수행 과정에서 부상을 입거나 사망할 수 있다.

#### 능력치/성장

- 능력치(ability): 힘, 민첩, 지능
- 성장치(초안): 50%, 30%, 30%
- 능력 테스트 계산에 능력치가 사용된다.
- 임무 완료 시 경험치를 획득한다.
- 레벨업 시 원하는 능력치 1을 올리고, 성장치 확률에 따라 추가 상승이 발생한다.
- 최대 레벨은 5이며, 레벨 구간별 필요 경험치는 증가한다.

#### 스탯

- 체력:
- 능력 테스트 실패 시 1 감소
- 0이 되면 해당 모험가 사망
- 최대값 존재
- 기력:
- 임무 완료 시 1 감소
- 기본값 4
- 1이면 피곤 상태(능력치 감소)
- 0이면 탈진 상태(임무 출전 불가)
- 영웅심:
- 능력 테스트 실패 시 체력 감소를 막고 자동 리롤을 유발하는 확률(%)

#### 특성

- 다키스트 던전 특성과 유사한 구조
- 임무 복귀 후 랜덤 획득 또는 교체 가능
- 긍정/부정 효과 모두 존재
- 긍정 최대 3개, 부정 최대 3개 보유 가능
- 긍정 특성은 최대 2개 고정 가능

#### 장비 귀속

- 모험가당 장비 슬롯은 2개
- 장비 착용 즉시 해당 모험가에 귀속
- 해고 전까지 귀속 해제 불가

#### 모집/휴식

- 모험가는 능력치/성장치/스탯/특성이 절차적으로 랜덤 생성된다.
- 매턴 일정 인원이 면접으로 유입되며, 막사 수용 범위 내에서 원하는 만큼 모집 가능하다.
- 해당 턴에 임무 미배치 모험가는 자동 휴식한다.
- 기본 휴식 효과: 체력 +1, 기력 +1

### 2) 왕국

- 안정도 스탯을 가진다.
- 안정도는 여러 경로로 증감한다.
- 안정도가 0이 되면 게임오버다.
- 안정도 최대값이 존재한다.

### 3) 임무

- 플레이어가 해결해야 하는 문제 단위
- 기한이 있으며, 제한 턴 내 미해결 시 실패 효과를 받는다.
- 인원 제한이 있어, 임무별 허용 인원 이하로만 배치 가능하다.
- 다수 능력 테스트를 모두 통과해야 임무 성공이다.
- 표시 정보: 능력 테스트, 보상, 기한, 실패 시 효과
- 임무 대응 실패 누적으로 안정도 0이 되면 게임오버다.
- 태그(예: 괴물, 질병)를 통해 상호작용 확장 가능하다.

### 4) 능력 테스트

- 구성: 요구 능력 + 난이도
- 요구 능력은 1개 이상 조합 가능(예: 힘+민첩)
- 계산: 임무 배치 모험가들의 해당 능력치 합을 사용
- 판정 방식:
- 모험가 측과 난이도 측이 각각 1~N 범위에서 롤
- 모험가 롤이 같거나 크면 성공
- 성공 시 다음 테스트로 진행
- 실패 시 임무 배치 모험가 전원 체력 1 감소
- 실패해도 재시도 가능
- 체력 0 모험가는 즉시 사망하며, 남은 인원으로 임무를 계속 진행
- 각 테스트마다 포기하기 가능(성공 보상 포기 후 임무 중단)

#### 영웅심 규칙

- 실패 시 모험가별 영웅심 테스트를 내부적으로 수행(UI 비표시)
- 한 명이라도 성공하면 실패해도 체력이 감소하지 않는다.
- 영웅심 발동 기회는 모험가당 임무당 1회다.

### 5) 장비

- 모험가에게 다양한 효과를 제공한다.
- 착용 후 해고 전까지 회수 불가다.

### 6) 스킬/소모품

- 세부 구상은 아직 없음(아이디어 기록 단계).

### 7) 시설

- 영구 보너스 제공 또는 모험가 상호작용용 시스템
- 다양한 방식으로 획득한다.

## 특성 아이디어(초안)

### 긍정

- 학생: 자신보다 레벨이 높은 모험가와 임무 진행 시 경험치 획득량 +50%
- 무한한 체력: 임무 종료 후 30% 확률로 기력 감소 없음
- 피학증: 부상 상태(체력 감소 상태)면 올스탯 +1
- 빠른 회복: 체력 회복량 +1
- 아드레날린: 체력 1일 때 올스탯 +3
- 리더: 자신이 가장 고레벨(동률 허용)일 때 올스탯 +1
- 모멘텀: 앞에서 성공한 능력 테스트마다 랜덤 스탯 +1(해당 임무 한정)

### 부정

- 외톨이: 같은 임무 배치의 다른 모험가 1명당 랜덤 스탯 -1(해당 임무 한정)
- 엄살쟁이: 부상 상태면 올스탯 -1
- 허약: 최대 기력 -1
- 결벽증: 질병 태그 임무 진입 시 올스탯 -2

## 시설 아이디어(초안)

- 병원: 모험가 슬롯 1칸, 매턴 할당 모험가 체력 +2
- 사창가: 골드 +2, 할당 모험가 기력 +2, 단 30% 확률로 기력 -2
- 스파: 할당 모험가 기력 +1
- 무기점: 할당 모험가가 다음 임무 1회 동안 올스탯 +1
